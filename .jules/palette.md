## 2024-05-24 - [CLI Visual Hierarchy Improvement]
**Learning:** Text-only CLI error messages can be difficult for users to scan, leading them to miss actionable next steps. Adding visual cues (like emojis for errors and tips) significantly improves scannability and UX without altering core logic.
**Action:** Always format CLI errors with a clear visual hierarchy (e.g., prepend '❌' to the error and '💡' to the actionable resolution step).
