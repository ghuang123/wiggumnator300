## 2024-04-03 - CLI Visual Hierarchy
**Learning:** In CLI applications, text error messages can be easily overlooked. Prefixing errors with '❌' and actionable resolutions with '💡' significantly improves scannability and reduces cognitive load.
**Action:** Always apply visual hierarchy with emojis (❌ for errors, 💡 for actionable steps) in CLI scripts to distinguish problems from solutions.
