## 2024-05-20 - Improve CLI error scannability with emojis
**Learning:** Actionable steps with clear visual hierarchy (e.g., emojis like ❌ for errors and 💡 for actionable hints) greatly improve scannability in CLI error states. It makes it immediately obvious what went wrong and how to fix it.
**Action:** Consistently use visual cues (like emojis) in CLI error messages and pair them with clear, actionable solutions.
