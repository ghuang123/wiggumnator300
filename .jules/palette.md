
## 2024-05-24 - CLI Error Message Scannability
**Learning:** In CLI-only environments without graphical interfaces, users often overlook terminal errors buried in walls of text. Emojis like ❌ and 💡 function as critical visual anchors to establish hierarchy and separate the failure state from the actionable resolution.
**Action:** Always prepend terminal error statements with a clear failure indicator (❌) and format resolution steps with an actionable indicator (💡) to improve scan time and reduce user frustration.
