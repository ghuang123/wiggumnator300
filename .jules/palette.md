## 2024-05-18 - Visual Hierarchy in CLI Errors
**Learning:** CLI applications lack standard UI components, making terminal output the primary user interface. Dense, unformatted text errors are hard to parse. Users miss actionable recovery steps when they blend in with standard error text.
**Action:** Always prepend clear visual markers like '❌' for errors and '💡' for actionable recovery steps in CLI scripts to significantly improve scannability and UX without graphical components.
